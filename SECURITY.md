# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in GitHub-to-EXE, we want to hear from you. Please report it responsibly by sending an email to gprebbiemensah@gmail.com with the subject "Security Vulnerability Report". Include the following details:
- A description of the issue.
- Steps to reproduce the vulnerability.
- Any potential impact.

## Response Process
- We will acknowledge your email within 48 hours.
- We will assess the vulnerability and work on a fix.
- We will notify you once the issue is resolved or mitigated.

## Disclosure Policy
We aim to fix vulnerabilities promptly and may release a security advisory via GitHub once resolved. Please do not disclose vulnerabilities publicly until a fix is available.

For more information, see [GitHub's security documentation](https://docs.github.com/en/code-security).
