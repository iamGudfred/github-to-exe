# GitHub-to-EXE

Convert public GitHub Python repositories into Windows executables.

## ⚠️ WARNING
This tool compiles third-party code. Only use repositories you trust.

## Setup
1. Run `scripts/setup.bat` (Windows)
2. Activate virtual environment: `call venv\Scripts\activate`
3. Start server: `python -m backend.app`
4. Open http://localhost:5000
